# GenoTSP
Genetic algorithm implementation to solve the famous NP-hard problem - The Travelling Salesman

<h3> Steps </h3>
1. Install requirements by running pip install -r requirements.txt <br>
2. Set the values in settings.py <br> 
3. Run genoTSP.py <br>
